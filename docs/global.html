Html

Parse an HTML string and get the outerHTML of each tag in it.

Type:
  • string
Properties:
Name Type Description
text string

OuterHTML of the tag.
name TagName

Name of the tag in lower case.
nestlevel number

Nest level of the tag. If it's not part of another tag, the value is 0.
selfclosing boolean

Whether the tag is closed by itself.
index Object

Indexes of the html tag in the input string. The end index is 'characters up to or not including', so the relevant tag can be extracted from the input string by "input.slice(index.start, index.end)".

TemplateArgument

Parse templates in wikitext. Templates within tags that prevent transclusions (i.e. <!-- -->, nowiki, pre, syntaxhighlist, source) are not parsed.

Type:
  • string
Properties:
Name Type Description
text string

The whole text of the template
name TemplateName

The name of the template. The first letter is always in upper case.
arguments Array.<TemplateArgument>

The arguments of the template
nestlevel number

The nestlevel of the template (0 if not embedded in other templates)
text string

The whole text of the template argument (e.g. 1=Wikipedian)
name string

The name of the template argument (e.g. 1)
value string

The value of the template argument (e.g. Wikipedian)
